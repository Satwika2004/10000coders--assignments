# List Built-in Functions in Python 

This folder contains programs based on list operations in Python.  
Problems are solved using Python built-in functions and simple loops.

## Problems:

1. Find the largest element in a list  
2. Remove duplicate elements from a list  
3. Reverse the elements of a list without using the built-in `reverse()` method  
4. Find the second largest element in a list  
5. Count how many even and odd numbers are present in a list
