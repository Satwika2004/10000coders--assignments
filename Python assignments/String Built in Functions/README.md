**String_BuiltIn_Functions**

This folder contains Python programs demonstrating the internal working of commonly used string built-in functions like `swapcase()`, `strip()`, `replace()`, and `rfind()`.  

## Functions Implemented
1. swapcase() – Convert uppercase letters to lowercase and vice versa.  
2. strip() – Remove leading and trailing spaces from a string.  
3. replace() – Replace occurrences of a substring with another substring.  
4. rfind() – Find the last occurrence of a substring in a string.
5. isupper() – Check if all characters in string are uppercase.
6. startswith() – Check if string starts with a given substring
7. endswith() – Check if string ends with a given substring
8. istitle() – Check if string is in title case
9. split() - Breaks a string into parts using a separator (default = space)
10. join() - Combines elements of a list into a single string using a separator.
11. center() - Places string in center by adding spaces (or characters) on both sides.
12. zfill() - Adds zeros (0) at the beginning of a string
