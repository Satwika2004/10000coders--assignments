# Feb 10 - Operators Assignment

This folder contains Python programs based on operators.

## Problems:
1. Swap two numbers without using a third variable  
2. Check whether a number is positive, negative, or zero  
3. Calculate area of rectangle and check whether it is a square  
4. Calculate Simple Interest  
5. Perform bitwise operations (AND, OR, XOR, left shift, right shift)  

All programs are implemented using Python.
